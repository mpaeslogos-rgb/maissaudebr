# maisaudebr

Repositório principal do projeto Mais Saúde BR.

## Estrutura

- `backend/` - servidor Node.js/TypeScript com Fastify e Prisma
- `frontend/` - aplicação Next.js

## Como rodar

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
